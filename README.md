# Descrição do Desafio módulo 2 - Criando um Relatório de Vendas Elegante com Power BI

Neste projeto foi criado um relatório no Power BI Desktop com base na sample Financials disponibilizada pela própria Microsoft. Os dados e outros elementos utilizados, mais o relatório gerado e o printscreen da tela no Power BI Service foram incluídos nesse repositório, como abaixo:

- 'Financial Sample.xlsx': base dde dados;
- 'logo_dio.png': imagem;
- 'clean blue.png': imagem;
- 'printscreen_relatorio criativo_powerbi service.pdf': prints das telas da área logada do Power BI Service, com o relatório publicado;
- 'relatorio_criativo.pbix': relatório gerado a partir das premissas do desafio.
